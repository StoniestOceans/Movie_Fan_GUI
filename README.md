# Movie Fan Generative UI Dashboard

## Overview
A generative UI dashboard for movie fans to explore facts, relationships, and trivia. This project leverages agentic AI to curate and present information dynamically.

## Tech Stack
- **Data Layer**: MongoDB Atlas (Vector Search & Knowledge Graph)
- **AI Orchestration**: Fireworks AI (Agent Router)
- **NLP**: NVIDIA NeMo (Conversational Intelligence)
- **Data Sources**: Wikipedia, Fanart.tv
- **Commerce**: Fandango (via x402 integration)
- **Frontend**: Next.js (React) + Tailwind CSS

## Architecture
The system uses a "Router Agent" pattern where a user's high-level intent is broken down into sub-tasks (e.g., "Find movies with Tom Hanks" -> Data Ingestion Agent -> Database -> UI Renderer).

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- MongoDB Atlas Account
- Fireworks AI API Key
- NVIDIA NeMo Framework
